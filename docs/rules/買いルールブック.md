---

## 1. 基本方針

| 項目 | ルール |
|------|--------|
| 注文方法 | **成行注文のみ**（指値で悩まない） |
| 単位 | **1株**（例外なし） |
| 頻度 | **1日最大2銘柄まで**（衝動買い防止） |
| 月間予算 | 月 _____ 円を上限とする（自分で設定） |
| 買い時間 | 前日夜 or 朝に注文 → S株は翌営業日の寄付で約定 |

## 2. 買い判定フローチャート

```
監視リストの銘柄を確認
    │
    ├─ スコア 70点以上か？ ──No──→ 買わない
    │       │
    │      Yes
    │       │
    ├─ 待ちシグナル出てないか？ ──Yes──→ 待つ（決算前・急騰後）
    │       │
    │       No
    │       │
    ├─ 買いシグナル出てるか？ ──No──→ 積極的には買わないが
    │       │                          月間予算に余裕あれば買ってOK
    │      Yes
    │       │
    ├─ 同業種3銘柄以上保有？ ──Yes──→ 他の業種を優先
    │       │
    │       No
    │       │
    ├─ 月間予算残ってる？ ──No──→ 来月まで待つ
    │       │
    │      Yes
    │       │
    └─ → **1株 成行で買う**
```

## 3. 買いシグナル詳細

### 3-1. 「押し目買い」ルール（最重要）

| 条件 | 判定基準 | 強さ |
|------|----------|------|
| 25日移動平均線割れ | 株価 < 25MA | ★★☆ |
| 75日移動平均線割れ | 株価 < 75MA | ★★★ |
| 52週安値から10%以内 | 株価 < 52週安値 × 1.10 | ★★★ |
| 前日比 -3%以上の下落 | 市場全体の下げか個別要因か確認 | ★★☆ |
| 月間 -10%以上の下落 | 業績に問題なければ絶好の買い場 | ★★★ |

**重要: 下落理由の確認**
- 市場全体の下げ（日経平均も下落）→ 気にせず買う
- 業種全体の下げ → 気にせず買う
- 個別の悪材料 → 内容を確認してから判断（下記参照）

### 3-2. 個別悪材料の判断基準

| 悪材料 | 対応 |
|--------|------|
| 一時的な費用計上（特別損失） | → 買い（本業に影響なし） |
| 為替差損 | → 買い（一時要因） |
| 通期据え置き（上方修正なし） | → 買い（悪材料ではない） |
| 下方修正（売上-5%以内） | → 様子見1週間 |
| 下方修正（売上-10%超） | → 見送り（次の決算まで待つ） |
| 減配発表 | → **買わない**（撤退ルール参照） |
| 不祥事・法令違反 | → **買わない**（回復に年単位かかる） |

### 3-3. 買い増しルール（既保有銘柄）

| 保有数 | 追加買いの条件 |
|--------|---------------|
| 1〜4株 | スコア70点以上 + 買いシグナルあり → OK |
| 5〜9株 | スコア80点以上 + 押し目（25MA割れ）→ OK |
| 10〜19株 | スコア85点以上 + 強い押し目（75MA割れ）→ OK |
| 20株以上 | 原則追加しない（分散を優先） |

## 4. 待ちシグナル（買ってはいけない時）

| 状況 | ルール |
|------|--------|
| 決算発表2週間前 | 発表後に判断する |
| 直近5日で+10%以上 | 過熱。冷めるまで最低1週間待つ |
| 52週高値の95%以上 | 高値掴みリスク。押し目を待つ |
| 市場全体がパニック的上昇 | 落ち着くまで待つ（FOMO禁止） |
| 自分が「今すぐ買わなきゃ」と焦っている | **絶対買わない**（感情シグナル） |

---

## 5. 月間予算管理

### 5-1. 予算配分の考え方

```
月間予算: _____ 円

配分:
├─ 新規銘柄（未保有）: 60%
├─ 既保有の買い増し:   30%
└─ 予備（暴落時用）:   10%
```

### 5-2. 暴落時の特別ルール

| 日経平均の下落幅 | 対応 |
|-----------------|------|
| -5%以上（月間） | 予備枠を解放。通常の1.5倍まで買いOK |
| -10%以上（月間） | 翌月予算の前借りOK（翌月は減額） |
| -20%以上（月間） | 年間で最大の買い場。可能な範囲で積極購入 |

**暴落時こそ淡々と買う。これが1株積立の最大の強み。**

## 6. 買ってはいけない銘柄（絶対NG）

| # | NG条件 | 理由 |
|---|--------|------|
| 1 | SNSやニュースで話題の銘柄 | 既に織り込み済み。高値掴みリスク |
| 2 | 自分が事業内容を説明できない銘柄 | 理解できないものは持てない |
| 3 | 「〇〇関連銘柄」というテーマだけで選んだ | テーマ終了で暴落する |
| 4 | 配当利回り6%超の異常高利回り | 株価暴落or減配の前兆の可能性 |
| 5 | IPO後1年以内 | ロックアップ解除で需給悪化リスク |
| 6 | 友人・知人に勧められただけの銘柄 | 自分のルールで判断する |
