## 1. 投資方針（前提）

| 項目 | 方針 |
|------|------|
| 投資スタイル | バランス型（配当＋業績成長の両立） |
| 買い方 | SBI証券 S株（ミニ株）で1株ずつ積立 |
| 保有期間 | 長期ガチホ（基本売らない） |
| 配当基準 | 2%以上あればOK（成長率があれば許容） |
| 監視規模 | 100銘柄以上を網羅的にスクリーニング |
| 目標 | 淡々と買い続けて配当を積み上げ、保有すること自体を楽しむ |

---

## 2. スクリーニングの全体像

```
[Step1: 足切りフィルタ]  母集団（東証全銘柄）→ 候補 200〜300銘柄
        ↓
[Step2: スコアリング]    候補 → 上位100〜150銘柄（監視リスト）
        ↓
[Step3: タイミング判定]  監視リスト → 今日のおすすめ 3〜5銘柄
        ↓
[Step4: 買い実行]        SBIでS株注文（手動）
```

---

## 3. Step1: 足切りフィルタ（不適格銘柄の除外）

### 3-1. 絶対除外条件（これに該当したら候補にしない）

| # | 条件 | 理由 |
|---|------|------|
| 1 | 配当利回り 2.0%未満 | 最低ラインを下回る |
| 2 | 赤字（営業利益が直近2期連続マイナス） | 配当原資がない |
| 3 | 自己資本比率 20%未満 | 財務不安で長期保有に不向き |
| 4 | 上場5年未満 | 実績不足で判断できない |
| 5 | 時価総額 300億円未満 | 流動性リスク・倒産リスク |
| 6 | 過去5年以内に無配転落歴あり | 配当の信頼性が低い |

### 3-2. 業種除外（任意・好みで調整）

| 除外候補 | 理由 |
|----------|------|
| 不動産（REIT除く） | 景気感応度が高い |
| 証券 | 市況依存が激しい |
| バイオ | 無配・赤字が多い |

※ 好みで追加・削除してOK

## 4. Step2: スコアリング（100点満点）

### 4-1. 配当の質（35点）

長期保有の幸福度に直結するため最重要。

| 指標 | 配点 | 採点基準 |
|------|------|----------|
| 連続増配年数 | 12点 | 0年=0点, 5年=6点, 10年以上=12点 |
| 過去10年の減配回数 | 10点 | 0回=10点, 1回=7点, 2回=4点, 3回以上=0点 |
| 配当性向 | 8点 | 30-50%=8点, 50-60%=5点, 60-70%=2点, 70%超=0点 |
| 配当利回り | 5点 | 2-3%=2点, 3-4%=4点, 4%以上=5点 |

**配当性向の考え方:**
- 30-50% → 余力たっぷり。増配余地あり。理想
- 50-60% → 標準的。問題なし
- 60-70% → やや高い。業績下振れ時に減配リスク
- 70%超 → 危険水域。利益の大半を吐き出している

### 4-2. 業績の成長力（30点）

ガチホの含み益＝幸福度。成長なき配当は先細り。

| 指標 | 配点 | 採点基準 |
|------|------|----------|
| 売上高成長率（3年CAGR） | 10点 | 0%以下=0点, 0-5%=4点, 5-10%=7点, 10%超=10点 |
| 営業利益成長率（3年CAGR） | 10点 | 同上 |
| ROE | 5点 | 5%未満=0点, 5-8%=2点, 8-12%=4点, 12%超=5点 |
| 営業利益率 | 5点 | 5%未満=0点, 5-10%=2点, 10-15%=4点, 15%超=5点 |

**CAGR（年平均成長率）の計算:**
```
CAGR = (直近期の値 / 3年前の値)^(1/3) - 1
```

### 4-3. 財務健全性（20点）

ガチホで安心して眠れるか。

| 指標 | 配点 | 採点基準 |
|------|------|----------|
| 自己資本比率 | 8点 | 20-40%=2点, 40-60%=5点, 60%超=8点 |
| 有利子負債/自己資本比率（D/E） | 6点 | 1.0超=0点, 0.5-1.0=3点, 0.5未満=6点 |
| フリーCF（直近3年連続プラス） | 6点 | 3年連続=6点, 2年=3点, 1年以下=0点 |

### 4-4. 割安度（15点）

同じ良い銘柄なら安く買いたい。

| 指標 | 配点 | 採点基準 |
|------|------|----------|
| PER（業種平均比） | 8点 | 業種平均の1.2倍超=0点, 0.8-1.2倍=4点, 0.8倍未満=8点 |
| PBR | 4点 | 2.0超=0点, 1.0-2.0=2点, 1.0未満=4点 |
| ミックス係数（PER×PBR） | 3点 | 22.5超=0点, 11.25-22.5=1点, 11.25未満=3点 |

**ミックス係数（グレアム指数）:**
- ベンジャミン・グレアムの基準: PER×PBR ≦ 22.5 なら割安
- 11.25以下なら超割安

## 5. Step3: タイミング判定（買い時チェック）

スコア上位でも「今日買うべきか」は別問題。

### 5-1. 買いシグナル（加点）

| シグナル | 判定方法 |
|----------|----------|
| 押し目 | 株価が25日移動平均線を下回っている |
| 52週安値圏 | 52週高値から20%以上下落 |
| 権利確定接近 | 権利確定日まで1〜2ヶ月以内 |
| 決算好調 | 直近四半期が通期進捗率で上振れ |

### 5-2. 待ちシグナル（様子見）

| シグナル | 判定方法 |
|----------|----------|
| 決算直前 | 決算発表まで2週間以内 → 発表後に判断 |
| 急騰直後 | 直近5日で10%以上上昇 → 冷めてから |
| 高値圏 | 52週高値の95%以上 → 押し目を待つ |

### 5-3. 分散チェック

| ルール | 内容 |
|--------|------|
| 同一業種上限 | 同業種で3銘柄以上保有中なら優先度↓ |
| 既保有銘柄 | 10株以上保有済みなら新規買い優先度↓ |

## 6. データソースと取得方法

### 6-1. 無料＋API不要のデータソース

| データ | おすすめソース | 取得方法 |
|--------|---------------|----------|
| PER, PBR, 配当利回り, 時価総額 | Yahoo!ファイナンス | 手動 or スクレイピング |
| 連続増配年数, 配当推移 | irbank.net | 手動 or スクレイピング |
| 業績推移, 四季報要約 | 株探（kabutan） | 手動 or スクレイピング |
| 決算スケジュール | 日本取引所（JPX） | CSV無料DL |
| 株価チャート・移動平均 | TradingView（無料版） | 目視確認 |
| スクリーニング初期候補 | SBI証券の銘柄スクリーナー | ブラウザ操作 |

### 6-2. SBI証券スクリーナーの活用

SBI証券の「銘柄スクリーニング」で足切りフィルタ（Step1）相当が設定可能:

```
設定例:
- 市場: 東証プライム / スタンダード
- 配当利回り: 2.0%以上
- PER: 5〜25倍
- 自己資本比率: 20%以上
- 時価総額: 300億円以上
- 営業利益: プラス
```

→ ここで絞った候補をCSVエクスポート or お気に入り登録

## 7. 運用フロー

### 7-1. 初期セットアップ（1回だけ）

```
1. SBI証券スクリーナーで足切り条件を設定
2. 候補200〜300銘柄を抽出
3. 監視銘柄リスト（CSV）に記録
4. 各銘柄のスコアリングを実施（irbank・株探で調査）
5. 上位100〜150銘柄をSBIお気に入りに登録
```

### 7-2. 日次ルーティン（5〜10分）

```
1. SBIお気に入りを開く
2. 前日比で大きく下がった銘柄をチェック
3. 監視リストの中で買いシグナルが出ている銘柄を確認
4. 条件が揃った銘柄があれば1株購入
5. 購入記録を更新
```

### 7-3. 週次レビュー（30分）

```
1. 保有銘柄の週間パフォーマンス確認
2. 来週の決算発表スケジュール確認
3. 新たに買いシグナルが出そうな銘柄をピックアップ
```

### 7-4. 月次メンテナンス（1〜2時間）

```
1. スコアリングの更新（四半期決算反映）
2. 監視リストの入れ替え（新規追加・除外）
3. 業種別の保有バランス確認
4. 配当受取実績の記録
```

### 7-5. 四半期イベント（決算シーズン）

```
1. 保有銘柄の決算内容をチェック
2. 減配発表 → 要注意フラグ
3. 増配発表 → スコア加点
4. 業績下方修正 → 継続保有判断
```
