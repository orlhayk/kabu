> 1株ずつ積み立てるからこそ、全体のバランスを意識する。
> 気づいたら特定業種に偏っていた、を防ぐためのルール。

---

## 1. 分散の基本方針

| 項目 | ルール |
|------|--------|
| 目標銘柄数 | 30〜50銘柄に分散 |
| 1銘柄あたり上限 | 保有総額の5%以内 |
| 1業種あたり上限 | 保有総額の20%以内 |
| 最低保有業種数 | 5業種以上 |

---

## 2. 業種分類と目安配分

| カテゴリ | 含まれる業種例 | 目安配分 |
|----------|---------------|----------|
| 通信・IT | 通信, 情報通信, ソフトウェア | 15-25% |
| 金融 | 銀行, 保険, リース, その他金融 | 15-25% |
| インフラ・素材 | 電力・ガス, 化学, 鉄鋼 | 10-20% |
| 消費・サービス | 食品, 小売, サービス | 10-20% |
| 製造 | 機械, 電気機器, 輸送用機器 | 10-20% |
| 商社・卸売 | 総合商社, 専門商社 | 5-15% |
| その他 | 建設, 不動産, 医薬品 | 5-15% |

※ 厳密に守る必要はないが、1カテゴリが30%を超えたら要リバランス

---

## 3. リバランスルール

### 3-1. チェックタイミング

- **月次**: 業種別比率を確認
- **四半期**: 銘柄別比率を確認

### 3-2. リバランス方法

**原則: 売らずに買いで調整する**

```
偏りが出た場合:
├─ 多すぎる業種 → その業種の新規買いを一時停止
├─ 少なすぎる業種 → 優先的にその業種から銘柄をピック
└─ 1銘柄が5%超 → その銘柄の買い増しを停止
```

売却によるリバランスは以下の場合のみ:
- 1銘柄が保有総額の10%を超えた（極端な偏り）
- 株価上昇ではなく他銘柄の下落で比率が上がった場合は売らない

---

## 4. 新規銘柄追加の判断基準

| チェック項目 | 基準 |
|-------------|------|
| 既に同業種で何銘柄持っているか | 3銘柄以上 → 他業種を優先 |
| 監視リストのスコア | 70点以上であること |
| 保有銘柄数の上限 | 50銘柄を超える場合は慎重に |
| 既存銘柄との重複 | 事業内容が似すぎていないか |

## 6. 長期目標の目安

| マイルストーン | 目標 |
|---------------|------|
| 年間配当 1万円 | まず最初の実感 |
| 年間配当 5万円 | 月のランチ代が配当で賄える |
| 年間配当 12万円 | 月1万円の不労所得 |
| 年間配当 30万円 | 中間目標。生活の一部を支える |
| 保有銘柄 30銘柄 | 分散がきいてくるライン |
| 保有銘柄 50銘柄 | 十分な分散。ここからは質の管理 |
