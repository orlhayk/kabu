> 基本はガチホ。だからこそ「売る時」を明確にしておく。
> ルールがないと、損切りできずに塩漬け or 利確を焦って手放すことになる。

---

## 1. 大原則

**「売らない」がデフォルト。以下に該当した場合のみ売却を検討する。**

含み損は気にしない。含み益も気にしない。
見るのは「この企業を今後も持ち続ける理由があるか」だけ。

---

## 2. 撤退（売却）トリガー

### 2-1. 即時撤退（発動したら速やかに売る）

| # | トリガー | 理由 |
|---|----------|------|
| 1 | **無配転落**（配当ゼロ発表） | 投資の前提が崩壊 |
| 2 | **粉飾決算・重大な不正会計** | 企業の信頼が根本から崩壊 |
| 3 | **上場廃止の決定** | 流動性がなくなる |
| 4 | **債務超過に転落** | 倒産リスクが現実化 |

### 2-2. 要観察→撤退検討（1四半期様子を見てから判断）

| # | トリガー | 判断基準 |
|---|----------|----------|
| 1 | **減配**（配当減額） | 一時的か構造的か確認。2期連続減配→撤退 |
| 2 | **2期連続の営業赤字** | 本業の稼ぐ力が失われた可能性 |
| 3 | **自己資本比率20%割れ** | 財務の安全域を下回った |
| 4 | **スコアが50点以下に低下** | 保有の前提条件を満たさなくなった |
| 5 | **事業内容が大きく変わった** | 自分が理解・共感できなくなった |

### 2-3. 判断フロー

```
トリガー発生
    │
    ├─ 即時撤退トリガー？ ──Yes──→ 翌営業日に成行売却
    │       │
    │       No
    │       │
    ├─ 要観察トリガー？
    │       │
    │      Yes
    │       │
    ├─ 監視リストに「要注意」フラグを立てる
    │       │
    ├─ 次の四半期決算まで待つ
    │       │
    ├─ 回復した？ ──Yes──→ フラグ解除。継続保有
    │       │
    │       No
    │       │
    └─ → **成行で全株売却**
```

## 3. 売らないルール（これでは売らない）

**以下の理由で売るのは禁止。感情に基づく売却を防ぐ。**

| # | 売ってはいけない理由 | なぜダメか |
|---|---------------------|-----------|
| 1 | 含み損が-20%を超えた | 株価は業績が良ければ戻る。損切りラインは「価格」ではなく「企業の質」で判断 |
| 2 | 他に買いたい銘柄ができた | 追加資金で買う。持ち株を売って乗り換えない |
| 3 | SNSで悲観論を見た | ノイズ。自分のスコアリングを信じる |
| 4 | 含み益が+50%を超えた | ガチホの前提。利確は配当で行う |
| 5 | 日経平均が暴落した | 市場全体の問題。個別企業の質は変わらない |
| 6 | なんとなく不安 | 「なんとなく」では売らない。具体的トリガーに該当するか確認 |

---

## 4. 部分売却ルール（例外的に使う場面）

| 状況 | 対応 |
|------|------|
| 1銘柄が保有総額の20%を超えた | 超過分を売却してリバランス |
| 生活資金が必要になった | 含み益が大きい銘柄から少額売却 |
| NISA枠の調整 | 特定口座→NISA口座への買い替え時のみ |

---

## 5. 売却時の手順

```
1. 撤退トリガーに該当するか再確認
2. 売却理由を購入記録に記入
3. 翌営業日に成行で売却
4. 監視銘柄リストから除外 or 「売却済み」にステータス変更
5. 売却代金は翌月の買い資金に充てる（出金しない）
```

---

## 6. 年次見直し

年1回（12月）に全保有銘柄を対象に以下を確認:

| チェック項目 | 基準 |
|-------------|------|
| スコアが60点以上を維持しているか | 60点未満→要観察リストへ |
| 配当が維持 or 増配されているか | 減配→撤退検討 |
| 自分がこの企業を説明できるか | 説明できない→そもそもなぜ持っているか再確認 |
| 業種の偏りが出ていないか | ポートフォリオ管理ルール参照 |
