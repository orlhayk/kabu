---

## 日次チェック（毎営業日 5〜10分）

- [ ] SBIお気に入りを開き、前日比を確認
- [ ] -3%以上下落した銘柄があるか確認
  - → あれば下落理由を確認（市場全体 or 個別要因）
  - → 買いルールに該当すれば1株購入
- [ ] 保有銘柄に重大ニュース（減配・不祥事）がないか確認
  - → あれば撤退ルールを参照
- [ ] 買った場合は購入記録.csvに記入

---

## 週次チェック（毎週末 20〜30分）

- [ ] 今週の購入銘柄を振り返り（衝動買いしていないか）
- [ ] 来週の決算発表スケジュールを確認（JPXサイト or 株探）
  - → 保有銘柄が決算2週間以内ならフラグ
- [ ] 監視リストの中で押し目（25MA割れ）の銘柄をピックアップ
- [ ] 来週の買い候補を2〜3銘柄決めておく
- [ ] 月間予算の残りを確認

---

## 月次チェック（月末 1〜2時間）

### 記録の更新
- [ ] 購入記録.csvが漏れなく記入されているか
- [ ] 配当が入金されていれば配当記録.csvに記入
- [ ] 保有状況サマリを更新（ポートフォリオ管理ルール参照）

### バランス確認
- [ ] 業種別比率を計算 → 偏りがないか
- [ ] 1銘柄が保有総額の5%を超えていないか
- [ ] 新規追加すべき業種はないか

### スコアリング
- [ ] 決算発表があった銘柄のスコアを更新
- [ ] スコア50点以下に落ちた銘柄がないか確認
  - → あれば要観察フラグ

### 翌月の計画
- [ ] 翌月の投資予算を設定
- [ ] 重点的に買いたい業種・銘柄をメモ

---

## 四半期チェック（3月・6月・9月・12月末 2〜3時間）

### 全銘柄レビュー
- [ ] 全保有銘柄のスコアを再計算
- [ ] 60点未満の銘柄をリストアップ → 撤退検討
- [ ] 監視リストの入れ替え（新規追加・除外）

### 配当実績
- [ ] 四半期の配当受取合計を計算
- [ ] 年間配当見込みを再計算
- [ ] 配当利回り（取得ベース）を更新

### ポートフォリオ分析
- [ ] 銘柄別の含み損益一覧を確認
- [ ] 業種別パフォーマンスを確認
- [ ] 全体の取得利回り vs 時価利回りを比較

### スクリーニング条件の見直し
- [ ] 足切りフィルタの条件は適切か
- [ ] スコアリングの配点バランスは適切か
- [ ] 新たに追加すべき指標はないか

---

## 年次チェック（12月 半日）

### 年間振り返り
- [ ] 年間の購入銘柄数・金額を集計
- [ ] 年間の配当受取額を集計
- [ ] 年間のパフォーマンス（含み損益 + 配当）を計算
- [ ] 売却した銘柄があれば、その判断は正しかったか振り返り

### 翌年の方針
- [ ] 月間予算の見直し
- [ ] 投資方針に変更があるか確認
- [ ] 重点的に狙う業種・テーマを設定
- [ ] スコアリング基準の改定が必要か検討

### 税務
- [ ] 特定口座の年間取引報告書を確認
- [ ] 確定申告が必要か確認（配当控除・損益通算）
- [ ] NISA枠の利用状況を確認

---

## 緊急時チェック（市場急変時）

### 日経平均 -5%以上の暴落時
- [ ] パニック売りしない（売らないルールを再読）
- [ ] 保有銘柄に個別の悪材料がないか確認
- [ ] 買いルールの暴落時特別ルールを確認
- [ ] 予算に余裕があれば、スコア上位銘柄を1〜2株購入
- [ ] 「暴落時こそ淡々と買う」を思い出す

### 保有銘柄に悪材料が出た時
- [ ] 即時撤退トリガーに該当するか確認
- [ ] 該当 → 翌営業日に売却。理由を記録
- [ ] 非該当 → 要観察フラグ。次の決算まで待つ
