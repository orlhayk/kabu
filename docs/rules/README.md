&#xFEFF;# 株スクリーニング自動化システム

## このフォルダは何か

個人の趣味として、日本の個別株を **SBI証券のS株（ミニ株）で1株ずつ淡々と積み立てる** ための判断支援システムです。投資助言ではありません。

## オーナーの投資スタイル（最重要）

```
方針:   配当＋ガチホで幸福度を上げる（売らない前提）
買い方: SBI証券 S株で「1株ずつ成行注文」
配当:   利回り2%以上あればOK（成長力があれば許容）
成長:   業績成長も重視するバランス型
売却:   原則しない。無配転落・粉飾など致命的事態のみ撤退
```

## ファイル構成と読む順番

| # | ファイル | 何が書いてあるか | いつ使うか |
|---|----------|-----------------|-----------|
| 1 | **設計書.md** | スクリーニングの全体設計。足切りフィルタ（6条件）とスコアリング（100点満点・4カテゴリ）の定義 | 銘柄を評価するとき |
| 2 | **買いルールブック.md** | 買い判定フローチャート。押し目の定義、待ちシグナル、月間予算管理、暴落時ルール | 「今日買うか？」を判断するとき |
| 3 | **売り・撤退ルール.md** | ガチホ前提での例外的な売却基準4つ。「売らない理由」も明文化 | 悪材料が出たとき |
| 4 | **ポートフォリオ管理ルール.md** | 分散ルール（1銘柄5%・1業種20%上限）、リバランス方法、長期目標 | 月次で全体を見るとき |
| 5 | **チェックリスト.md** | 日次5分〜年次半日の全運用タスク一覧 | 毎日〜毎年のルーティン |
| 6 | **監視銘柄リスト.csv** | スコアリング対象の全銘柄データ（PER/PBR/配当性向/成長率等） | スコア計算・更新時 |
| 7 | **購入記録.csv** | 売買の履歴（日付・銘柄・株数・理由） | 買った/売った時 |
| 8 | **配当記録.csv** | 配当の入金記録（銘柄・金額・累計） | 配当入金時 |

## スコアリング早見表（100点満点）

```
配当の質     35点  連続増配(12) + 減配なし(10) + 配当性向(8) + 利回り(5)
業績の成長力  30点  売上CAGR(10) + 営利CAGR(10) + ROE(5) + 営業利益率(5)
財務健全性   20点  自己資本比率(8) + D/Eレシオ(6) + FCF(6)
割安度       15点  PER業種比(8) + PBR(4) + ミックス係数(3)
```

## 買い判定の要約

```
スコア70点以上 → 買い候補
  + 25MA割れ or 52週安値圏 → 買いシグナル
  - 決算2週間前 or 急騰直後 → 待ち
  - 同業種3銘柄以上保有 → 他を優先
  → 条件OK + 月間予算あり → 1株 成行で買う
```

## AIアシスタントへの指示

このフォルダの内容について質問・相談された場合:

1. **上記のファイル構成を把握した上で回答してください**
2. 銘柄の評価を求められたら `設計書.md` のスコアリング基準に従ってください
3. 買い判断を求められたら `買いルールブック.md` のフローに従ってください
4. 売り判断を求められたら `売り・撤退ルール.md` のトリガーに該当するか確認してください
5. **投資助言はしないでください**。あくまでオーナーが定めたルールに基づく判断支援です
6. CSVファイルの更新を求められたら、既存のカラム構造を維持してください
