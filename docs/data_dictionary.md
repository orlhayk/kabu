# データ定義

## `data/watchlist.csv`

監視銘柄の評価元データです。

| 列名 | 用途 |
|---|---|
| コード | 証券コード |
| 銘柄名 | 表示名 |
| 業種 | 分散管理用 |
| 配当利回り(%) | 足切り・配当採点 |
| PER(倍) | 割安度採点 |
| PBR(倍) | 割安度採点 |
| ミックス係数 | `PER x PBR` |
| 自己資本比率(%) | 財務採点 |
| ROE(%) | 成長採点 |
| 営業利益率(%) | 成長採点 |
| 売上CAGR3年(%) | 成長採点 |
| 営利CAGR3年(%) | 成長採点 |
| 連続増配(年) | 配当採点 |
| 過去10年減配回数 | 配当採点 |
| 配当性向(%) | 配当採点 |
| D/Eレシオ | 財務採点 |
| FCF3年連続+ | 財務採点。`3`, `2`, `1`, `0` のような年数で扱う想定 |
| スコア合計 | 自動計算対象 |
| 最終更新日 | データ更新日 |
| メモ | 個別事情や一時要因 |

## `data/purchases.csv`

売買履歴です。

| 列名 | 用途 |
|---|---|
| 日付 | 約定日 |
| コード | 証券コード |
| 銘柄名 | 表示名 |
| 買/売 | `買` または `売` |
| 株数 | 約定株数 |
| 約定単価 | 1 株あたり価格 |
| 約定金額 | 総額 |
| 累計保有数 | 取引後残高 |
| 買い理由 | ルール上の理由 |
| メモ | 補足 |

## `data/dividends.csv`

配当入金履歴です。

| 列名 | 用途 |
|---|---|
| 入金日 | 配当入金日 |
| コード | 証券コード |
| 銘柄名 | 表示名 |
| 配当区分(中間/期末) | 中間または期末 |
| 1株配当(円) | 1 株あたり配当 |
| 保有株数 | 権利時点の株数 |
| 税引前(円) | 税引前受取額 |
| 税引後(円) | 税引後受取額 |
| 累計配当(円) | 累計管理用 |
| メモ | 補足 |

## `data/watchlist_live.csv`

朝バッチが出力する、当日価格と採点を含んだ展開版です。

主な追加列:

- 成行目安(円)
- 前日終値(円)
- 25MA(円)
- 75MA(円)
- 52週高値(円)
- 52週安値(円)
- 5日騰落率(%)
- 1ヶ月騰落率(%)
- 買いシグナル
- 待ちシグナル
- SBI候補判定
- 候補優先度

## `data/sbi_candidates.csv`

朝に確認するための候補一覧です。`watchlist_live.csv` から、候補優先度の高い銘柄を抜き出します。

## `data/sheets_candidates.csv`

Google スプレッドシート向けに列順を整理した一覧です。

主な列:

- 表示順
- アクション
- コード
- 銘柄名
- 成行目安(円)
- 差額(円)
- スコア
- 優先度
- 買いシグナル
- 待ちシグナル

## 推奨追加列

`watchlist.csv` に次を足すと、設計書どおりの判定に近づきます。

- 時価総額(億円)
- 上場年数
- 無配転落歴5年
- 直近2期営業赤字
- 25MA
- 75MA
- 52週高値
- 52週安値
- 決算予定日
- 権利確定日
- 保有株数
- ステータス
