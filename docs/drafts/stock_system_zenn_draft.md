## なぜ「配当＋ガチホ」にたどり着いたか

> 趣味の個別株ミニ投資（1株積立）を効率化するための仕組み  
> 目的: 配当＋ガチホで幸福度を上げる

日本株で長期投資をしようとすると、どうしても

- 株価の上下にメンタルを振り回される
- 一度決めたはずのルールを、相場が荒れた瞬間に破りがち

という問題にぶつかります。

そこで僕は、「**感情をできるだけコードに押し込める**」ことを目標に、

- どんな銘柄を持ちたいのか（投資哲学）
- それをどう数値化するか（スコアリング）
- いつ買うか／いつは買わないか（タイミング）

をすべて文章とコードに落とし込みました。

## システム全体の設計ざっくり

リポジトリ全体の設計は `docs/rules/設計書.md` に細かく書いてありますが、ざっくり図にするとこんな流れです。

```text
[Step1: 足切りフィルタ]  母集団（東証全銘柄）→ 候補 200〜300銘柄
        ↓
[Step2: スコアリング]    候補 → 上位100〜150銘柄（監視リスト）
        ↓
[Step3: タイミング判定]  監視リスト → 今日のおすすめ 3〜5銘柄
        ↓
[Step4: 買い実行]        SBIでS株注文（手動）
```

- 足切りフィルタでは「長期ガチホに向かない地雷銘柄」を機械的に除外
- 残った銘柄を **4カテゴリ・100点満点** でスコアリング
- 地合い（VIX＋日経平均）と個別チャートで「今日買うかどうか」を判断

という構造です。

## 4カテゴリ・100点スコアリングの設計

詳細は `docs/rules/設計書.md` に書いていますが、カテゴリだけ抜き出すとこうなります。

- **配当の質（35点）**
  - 連続増配年数 / 減配回数 / 配当性向 / 配当利回り
- **業績の成長力（30点）**
  - 売上・営業利益のCAGR / ROE / 営業利益率
- **財務健全性（20点）**
  - 自己資本比率 / D/Eレシオ / フリーCF
- **割安度（15点）**
  - PER（業種平均比）/ PBR / PER×PBR

配当の質に35点と最も大きなウェイトを置いているのは、

- 「長期ガチホの幸福度は、目先の株価より**配当の安心感**にかなり依存する」

と考えているからです。

## 他AIにルールを採点してもらった結果（要約）

このルール一式については、別のAIにもレビューしてもらいました。

> 株式投資ルール全体評価：A評価 / 92点  
> 個人投資家が実践するルールとして、極めて完成度が高い。  
> 「感情を排し、ルールに従って淡々と実行する」というシステム投資の理想形を  
> うまく言語化・構造化できている。

強みとして挙げられていたのは、だいたい次のようなポイントです。

- 投資哲学（配当＋ガチホ）が明確で、一貫している
- スコアリング設計が論理的で、実際のコードに落とし込みやすい
- 買い付けルール・リスク管理・撤退戦略まで含めた「投資システム」になっている
- 文書（ルールブックやチェックリスト）が整っていて、再現性が高い

一方で、改善ポイントとしては

- 業種平均PERとの比較など、一部はまだ「設計＞実装」の状態
- バックテストや過去事例の見せ方を工夫すると、説得力がさらに上がる

といった指摘がありました。

この記事では、こうした第三者レビューも踏まえつつ、**実際にどんな銘柄を「一生ガチホ候補」として選んだのか**を公開していきます。

## この記事の使い方と今後のアップデート

- この記事のスコアやコメントは、あくまで「2026年時点のざっくり評価」です  
- 実際の投資判断は、最新の決算・ニュース・株価水準を必ず確認する必要があります  
- 一方で、「**自分のルールで機械的にスクリーニングし、感情をコード側に寄せる**」という考え方は、今後も変わらない軸として使い続けられます

今後は、

- 業種平均PERなど、まだ手作業に近い部分の自動化
- 過去データを使った簡易バックテスト
- GitHub Actions を使った「毎朝8時に自動スクリーニングしてZenn用ドラフトを更新する」仕組み

なども試していく予定です。
